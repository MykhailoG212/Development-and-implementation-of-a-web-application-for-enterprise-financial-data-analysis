# Система аналізу фінансових даних

## Огляд проєкту

**Система аналізу фінансових даних** — це вебзастосунок для обліку, керування та аналізу фінансових потоків підприємства або приватної особи. Система побудована на мікросервісній архітектурі з використанням Docker.

Застосунок дозволяє користувачам реєструватися, вести облік доходів і витрат, а також автоматично отримувати аналітику у вигляді графіків та ключових фінансових показників.

## Технологічний стек

* **Backend:** Python 3, Flask, Flask-Login (автентифікація), Flask-PyMongo.
* **Database:** MongoDB (документоорієнтована БД).
* **Frontend:** HTML5, CSS3, JavaScript, Fetch API.
* **Visualization:** Chart.js.
* **DevOps & Server:** Docker, Docker Compose, Nginx, Gunicorn.

## Швидкий старт через Docker Compose. Це найкращий спосіб запуску, оскільки він автоматично налаштовує базу даних та мережеву взаємодію.

1. **Встановіть** [Docker Desktop](https://www.docker.com/products/docker-desktop) та Docker Compose.
2. **Запустіть систему** командою з кореня проєкту:
```bash
docker-compose up --build

```


3. **Очікуйте завершення збірки.** Буде піднято три сервіси:
* `mongodb_container`: База даних (порт 27017).
* `backend_container`: Flask API (внутрішній порт 5000).
* `frontend_container`: Nginx вебсервер (зовнішній порт 3000).


4. **Відкрийте застосунок у браузері:**
 [http://localhost:3000](http://localhost:3000)

## Основні можливості
1. **Автентифікація та безпека:**
* Реєстрація та вхід користувачів.
* Захист паролів (хешування).
* Приватний доступ (користувач бачить тільки свої дані).


2. **Управління фінансами (CRUD):**
* Створення записів (Дохід/Витрата).
* Перегляд історії транзакцій у таблиці.
* Видалення записів.
* Оновлення записів.

3. **Аналітика та візуалізація:**
* Автоматичний розрахунок загального прибутку та рентабельності.
* Інтерактивні графіки (Chart.js) для візуалізації динаміки доходів та витрат.
